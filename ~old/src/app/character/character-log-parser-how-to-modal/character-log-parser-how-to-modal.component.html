Log parser

Badge detection works by taking advantage of the in-game chat log file. When one of your characters receives a new badge or you select a badge title, a text line appears in the chat box. If you have chat logging enabled, the chat log will be saved to a file which can be uploaded to the log parsing tool. Chat logging is not enabled by default, but you can enable it via the options window as illustrated below:

Unfortunately, this will only detect newly earned badges, and not badges you have previously earned. To detect all badges, you will need to run the settitle binds.

Make sure that you log into your character after enabling chat logging so the logs contain a record of which character to associate the badge with. Simply log out and then back in after enabling log parsing to ensure the log contains your character name.

Follow the steps below to install and run the settitle binds

  1. Download the settitle.txt file to your City of Heroes directory.
  2. Copy and paste the following text into your chat box: /bind_load_file "C:\<CoHDirectory>\settitle.txt". Replace <CoHDirectory> with the name of your City of Heroes install directory.

If you've followed all steps correctly, your existing badges will be saved to your log file which you can upload into this log parsing tool.

You can find the log files in either the <CoHDirectory>/accounts/<username>/logs directory (if using the HC Launcher) or the <CoHDirectory>/<username>/logs directory (for other launchers). Select the appropriate log file(s) and upload them via the log parser dialog.
